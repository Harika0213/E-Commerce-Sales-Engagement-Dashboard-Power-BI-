
📊 Project Overview

This project involved transforming raw e-commerce data (orders, sessions, and marketing campaigns) into a professional, single-page executive dashboard. The goal was to provide a centralized view of business health, marketing efficiency, and customer behavior.

🚀 Key Features & Insights

Executive KPIs: Tracks Gross Revenue, Total Orders, Average Order Value (AOV), and Conversion Rate % to provide an instant pulse of the business.

Revenue Trend Analysis: A monthly time-series view showing growth patterns from 2023 to 2024.

Global Sales Distribution: Interactive map visualization showing revenue by geographic region.

Engagement Metrics: Analysis of Bounce Rate by Marketing Channel to identify which sources (Email, Social, Organic) drive the highest quality traffic.

Operational Details: Breakdown of Payment Methods and a granular Campaign Performance Table for ROI analysis.

🛠️ Technical Stack

Tool: Power BI Desktop

Data Modeling: Established relationships between Orders, Sessions, and Marketing Campaign tables.

DAX (Data Analysis Expressions): Used for creating custom measures such as Conversion Rate, AOV, and dynamic date formatting.

UI/UX Design: Implemented a dark-themed professional layout with a functional sidebar for filtering and navigation.

📂 How to Use

Download the .pbix file.

Open in Power BI Desktop.

Use the Source and Year slicers on the left sidebar to interact with the data and drill down into specific segments.
